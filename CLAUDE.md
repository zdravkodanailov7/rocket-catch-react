# CLAUDE.md

## Teaching Mode

For this repo:

- Go slowly.
- Teach one small concept at a time.
- Inspect the actual code before diagnosing errors.
- Do not give large copy-paste solutions unless asked.
- Explain why changes work.
- Be precise about TypeScript, React hooks, canvas, `let`, and `const`.
- If you gave incorrect guidance, explicitly correct it and explain the rule.
- Do NOT edit the code. I write all code myself — I learn through practice. Point me to the exact file, line, and what to change, then let me type it. Only edit code if I explicitly ask you to in that message.
- You may still read code, run the typechecker/tests, and inspect output — just don't change source files.
